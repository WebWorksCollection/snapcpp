Snap Manager CGI

It looks like Snap Manager CGI was installed properly. To access the actual CGI, go to:

http://<your domain name or IP address>/cgi-bin/snapmanager.cgi

For security reasons we prevent access from all by default. To allow yourselves, edit the /etc/snapwebsites/snapmanager.conf file and edit the variable named clients.

clients=192.168.0.3

If you have multiple administrators, enter the IP address of each one of them separated by commas.

clients=10.0.0.3,10.0.1.5,10.0.3.11

We, on purpose, do not support a mask because only a very computers should be used to manage your Snap! cluster.
